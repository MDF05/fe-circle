# Support

## Need Help?

If you have questions or run into issues, please use the following channels:

1.  **GitHub Issues**: For bug reports and feature requests.
2.  **Discussions**: For general questions and community interaction (if enabled).

## Creating a Bug Report

When creating a bug report, please include:

*   **Browser & Version**: (e.g., Chrome 100 on Windows 10)
*   **Steps to Reproduce**: Clear, numbered steps.
*   **Expected vs Actual Behavior**.
*   **Screenshots/Videos**: Visual proof of the issue.
*   **Console Errors**: Any errors visible in the DevTools console.
